# 🌐 Anusha Mall — Portfolio Website

A modern, responsive personal portfolio website showcasing my skills, projects, certifications, and contact information.

🔗 **Live Site:** [anushaxdd.github.io/MyResume](https://anushaxdd.github.io/MyResume/)

---

## ✨ Features

- **Dark Theme** — Sleek dark UI with blue-purple gradient accents
- **Glassmorphism Cards** — Frosted glass effect on all content cards
- **Scroll Animations** — Fade-in animations triggered on scroll
- **Fully Responsive** — Adapts seamlessly to mobile, tablet, and desktop
- **Smooth Navigation** — Fixed navbar with smooth scroll to sections
- **No Dependencies** — Pure HTML, CSS, and vanilla JavaScript

## 📋 Sections
